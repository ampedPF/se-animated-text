# SteamElements Animated Text

A StreamElements' custom widget to display 2 animated texts in rotation.  

1. Create an overlay from StreamElements' [Dashboard](https://streamelements.com/dashboard/overlays).

1. Add a `STATIC/CUSTOM > Custom widget`.

1. From the Settings panel, click on the `Open Editor` button.

1. Delete the content of the HTML tab.

1. Copy/Paste the content of `index.html` into the HTML tab.

1. Repeat the process for the `style.css` file into the CSS tab.

1. Repeat the process for the `script.js` file into the JS tab.

1. Repeat the process for the `fields.json` file into the FIELDS tab.

1. Click on the "Done" button at the bottom right.

1. Customize the widget using the other panels on the left side.
